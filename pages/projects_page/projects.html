Manpower Scheduling Board Web Application

Ah yes, this is my computer science degree capstone project where we were tasked with creating a scheduling board application for Horizon Electric Inc. to replace their manual "whiteboard and magnets" method of scheduling employees. We decided to make the project web based, as it would be easier to access using different types of devices and because my group were all familar with PHP development due to a previous course we all took.

The project used PHP, MySQL, and Docker for the backend, and HTML, CSS, and JavaScript (with jQuery) for the frontend. Surprisingly, this project didn't use any major libraries or frameworks even though the major workflow heavily emphasized drag and drop functionality and asynchronous updating of elemnts (copilot, please stop)
